📖 How to Use RajOS

Your complete productivity OS — built for students, creators and self-improvers. Here's how to get the most out of every tab.

1

⊞ Dashboard

Your daily command center. Switch between Today and Monthly views. Use the date navigator (‹ ›) to check any past day's productivity. Charts show task breakdown and weekly progress.

2

✓ Tasks

Add tasks via the + button — single or bulk. Tag tasks like #AI or #fitness for instant filtering. Set dates (Today/Tomorrow/This Month) and priorities. Edit any task with ✏.

3

◎ Habits

Track daily habits in Week view. Navigate weeks with ‹ ›. Switch to Month view to see a full calendar of each habit. Edit habits anytime with ✏. Streaks auto-calculate.

4

📚 Learning

Add any resource — book, course, YouTube, podcast, people you learn from. Set start/end dates to track how long you took. Filter by status and type. Progress slider updates instantly.

5

⏱ Pomodoro

Set 25-min focus sessions. Add your working tasks in the "Working On" list and click one to activate it. After 4 sessions a long break triggers automatically. All sessions log below.

6

📓 Journal

Daily MIT tasks, wins, challenges, gratitude and free notes. Use + Backdate to add entries for any past date. Export all entries as PDF. Auto-saves as you type.

7

🗄 Vault

Save any important link, note, tool or resource. Add tags for quick filtering. Edit inline by expanding any item. Your GitHub and Expense Tracker are pre-loaded.

8

+ FAB Button

The floating + button at bottom-right lets you quickly add a task, learning item, journal entry or vault item from any tab without navigating away.

☁ Google Drive Sync

Connect your Google Drive to automatically save and load your RajOS data. Uses Google Drive API — your data stays in your own Drive.

Get this from Google Cloud Console → APIs → Credentials → OAuth 2.0 Client ID (Web application). Add your page URL as authorized origin.
✓Connected —

⚙ Setup Instructions

1

Create Google Cloud Project

Go to console.cloud.google.com → New Project → Enable "Google Drive API".

2

Create OAuth Credentials

APIs & Services → Credentials → Create → OAuth 2.0 Client ID → Web Application. Add your current page URL as Authorized JavaScript Origin.

3

Paste Client ID above & Connect

Copy the Client ID (ending in .apps.googleusercontent.com), paste above, click Connect. Sign in with Google to authorize.

4

Save & Load

"Save to Drive" uploads rajos-data.json to your Drive. "Load from Drive" restores the latest backup. Data is saved in your Drive's app folder.
