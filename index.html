📖 How to Use RajOS v16

Your complete personal productivity OS — tasks, habits, goals, journal, vault, learning tracker, time manager and Pomodoro. All data lives in your browser. Here's how every feature works.

1

⊞ Dashboard

Your daily command center. Defaults to Today view — shows task list, habits, learning, timeline and a day summary. Use the ‹ › arrows to navigate to any past day. Switch to Month view to see task completion by day, habit performance, learning progress and journal entries across the whole month. Click ✏ Edit on the quote bar to manage your motivational quotes (format: Quote | AUTHOR).

2

✓ Tasks

Three dropdowns control what you see: Date (All Time / Today / Yesterday / Tomorrow / This Week / This Month / Pick Date), Status (All / Pending / Done), and Priority (All / High / Medium / Low). Done tasks show with strikethrough in the same list when Status = All or Done. Recurring tasks — set a repeat frequency (Daily / Weekdays / Weekly / Monthly) when adding a task and it auto-spawns each day. A 🔁 badge appears on recurring tasks. Use 🧮 Matrix view to see tasks in the Eisenhower grid — drag between quadrants or use the Q1/Q2/Q3/Q4 selector on each task. Tag tasks like #AI or #fitness — each tag gets a consistent color.

3

◎ Habits

Month Grid — each row is one of your habits, each column is a day. Click any past cell to mark done (✕) or undo. Today is highlighted in blue. Streak 🔥, progress % and monthly count shown per habit. Use ⠿ Reorder to drag habits into your preferred order. Click 📊 Analytics to see: streak calendar per habit (use the dropdown), best/worst days of the week, and a 70-day heatmap per habit. All analytics read from your actual habit log — only habits you've added appear here.

4

🎯 Goals

Add Monthly or Quarterly goals with a category, target date and progress slider. Link a habit to a goal to track them together. Use the Eisenhower Matrix tab inside Goals to view all active tasks in a 2×2 urgent/important grid. Use 🌅 Daily Planning to set your 3 MITs (Most Important Tasks) each morning — they auto-create as tasks and show on the Dashboard.

5

📚 Learning

Track books, courses, YouTube channels, podcasts, people you learn from, or any skill. Set start/end dates to track duration. Adjust progress with the slider — hitting 100% auto-marks it Done. Filter by status (Active / Queue / Done / Archived) and type. Export your list as CSV.

6

📓 Journal

Daily entries with MIT tasks, Wins, Challenges, Gratitude and free Notes. Set your mood with the emoji bar. Entries auto-save as you type. Use + Backdate to add an entry for any past date. Export all entries as a PDF. The journal streak and avg mood show in the stat bar.

7

🗄 Vault

Save important links, notes, tools and resources. Click 📌 on any item to pin it — pinned items always appear at the top with a gold highlight. Filter by category or search by title, note or tag. Expand any item (▼) to edit inline. All tags become clickable filters in the sidebar.

8

⏰ Time Manager

Weekly calendar grid — each slot is one hour. Click any slot to open the log modal pre-filled with that date and hour. Completed tasks with a start time and pomodoro sessions auto-appear. Use the + Quick Log sidebar to log by category (Deep Work, Sleep, Leisure, etc.). The Week Split donut shows your time breakdown.

9

⏱ Pomodoro

Click ⏱ Pomo in the top bar to open the floating timer. Drag it anywhere on screen or go fullscreen with ⛶. Choose from 5 timer styles: Ring, Stopwatch, Flip, Dots, Clock. Add focus tasks to track what you're working on. Session counts and minutes are saved daily and shown on the Dashboard.

10

+ FAB Quick-Add

The floating + button (bottom-right) opens a quick-add menu from any tab: Task, Habit, Learning, Journal Entry, Vault item, or Goal — no need to navigate away.

11

☁ Google Drive Sync

Settings → Drive Sync → paste your Google OAuth Client ID → Connect. Use Save to Drive to upload your data as rajos-data.json to your Google Drive. Use Load from Drive to restore it on any device.

12

💾 Export & Backup

Click the ⬇ export icon (top-right) for: Full JSON backup, Tasks CSV, Learning CSV, Journal PDF, or PDF daily report. Use Settings → Data → Import to restore from a JSON backup. Export regularly — all data is local to your browser.

☁ Google Drive Sync

Connect your Google Drive to automatically save and load your RajOS data. Uses Google Drive API — your data stays in your own Drive.

Get this from Google Cloud Console → APIs → Credentials → OAuth 2.0 Client ID (Web application). Add your page URL as authorized origin.

⚙ Setup Instructions

1

Create Google Cloud Project

Go to console.cloud.google.com → New Project → Enable "Google Drive API".

2

Create OAuth Credentials

APIs & Services → Credentials → Create → OAuth 2.0 Client ID → Web Application. Add your current page URL as Authorized JavaScript Origin.

3

Paste Client ID above & Connect

Copy the Client ID (ending in .apps.googleusercontent.com), paste above, click Connect. Sign in with Google to authorize.

4

Save & Load

"Save to Drive" uploads rajos-data.json to your Drive. "Load from Drive" restores the latest backup. Data is saved in your Drive's app folder.
